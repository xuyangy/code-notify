# Security Policy

## Supported Versions

Only the latest released version of Code-Notify is actively supported with security fixes.

## Reporting a Vulnerability

Please do not open a public issue for a vulnerability.

Report security issues privately through GitHub Security Advisories:

https://github.com/mylee04/code-notify/security/advisories/new

Include:

- Affected version
- Operating system
- Install method
- Reproduction steps
- Impact and any known workaround

We will review reports as quickly as possible and coordinate a fix before public disclosure when needed.
