Feature release with Antigravity approval banners, a limit-reached badge, and
configurable notification wording, on top of `v2026.07.1`.

## Highlights

### Antigravity MCP approval banners and lifecycle hooks

- Antigravity MCP tool calls that pause for approval now banner, resolving
  `mcp(<server>/<tool>)` permission rules (with `*` wildcards) the same way
  `command(...)` rules already worked; unlisted tools default to `ask`.
- Adopted agy 1.1.3 lifecycle hooks (`PreInvocation`/`Stop`) for turn-start and
  turn-end signals: the running indicator now lights at prompt submission
  (including turns that never call a tool) and a `Stop` payload carrying a
  terminal error now maps to a failure alert instead of "task complete".
- Fixed `Stop` and other lifecycle hooks being silently rejected by agy 1.1.3,
  which requires flat handler lists instead of the grouped
  `{"hooks": [...]}` shape.

### Usage-limit badge and spinner reliability

- Added a `StopFailure` hook (bash and Windows) so the tmux running indicator
  no longer spins forever after Claude stops and waits for a rate limit to
  reset; a rate-limited turn now pauses the indicator and badges the window
  with an hourglass "Limit Reached" alert instead.
- Fixed a spinner/badge overlap flicker on tmux transitions, where the old
  event badge could briefly repaint next to the new running-indicator
  spinner.
- Refreshed the usage-limit alert wording.

### Configurable notification wording

- Added a `cn wording` command with short/long phrasing pools for
  completion, idle, permission, MCP elicitation, input, and question
  events, plus independent banner/voice style controls
  (`CODE_NOTIFY_BANNER_WORDING` / `CODE_NOTIFY_VOICE_WORDING` overrides).
  Mirrored on Windows.
- Voice alerts now speak the project name (e.g. "in project code-notify"),
  with independent `cn wording project banner|voice` toggles; the TTS cache
  is now scoped per project.
- Cached TTS audio now updates its modified time on playback, so
  time-based pruning (`find -mtime`) reflects last use, not creation.

### Compatibility and cosmetic fixes

- Suppressed hooks replayed by opencode compatibility plugins
  (oh-my-openagent), which could start the tmux spinner with no matching
  turn-end to clear it.
- The team task-completion event now badges with 🏅 instead of reusing the
  generic task-complete icon.
- Reworded the idle notification to "can take more work now".
