This is the first CalVer release of the fork. It collects the fork-specific work
since upstream `v1.10.0`.

## Highlights

### Antigravity CLI support

- Added end-to-end support for Google Antigravity CLI (`agy`) through an
  installed plugin.
- Added permission-request and error notifications plus debounced task-complete
  inference for a CLI that does not yet expose a working stop hook.
- Hardened completion detection with tmux pane-settle gating, stale-token
  cleanup, quote normalization, scoped tool parsing, and paths-with-spaces
  support.

### Codex and Claude hook improvements

- Migrated Codex integration from the legacy `notify` setting to lifecycle hooks
  in `~/.codex/hooks.json`.
- Disabled duplicate Codex TUI notifications while Code-Notify owns delivery and
  improved preservation and repair of existing configuration.
- Added immediate Claude approval alerts through `PermissionRequest`, preserved
  question notifications, and improved prompt/subagent lifecycle messages.

### tmux workflow awareness

- Added click-to-focus so a notification can jump to the originating tmux
  window and pane.
- Added event badges to tmux window names, prompt/focus-based clearing, and a
  setting controlling whether the currently visible window is badged.
- Added running-state indicators with an optional animated status-line spinner.
- Added approval-resume detection, process-exit cleanup, Codex pane-settle
  completion, and synthetic idle reminders for agents without native idle
  events.
- Hardened badge and spinner behavior around renamed windows, quoted paths,
  visible windows, agent exits, and hook-less completion paths.

### Notification delivery and voice

- Added persistent alert types, timed snooze, and ntfy phone delivery alongside
  Slack and Discord.
- Added event-specific rotating messages and clearer idle, input-request, and
  subagent notifications.
- Added ElevenLabs voices, environment-variable API keys, synthesized-audio
  caching, test/configuration commands, and fallback to the native voice engine.

### Performance and reliability

- Reduced notification latency and detached post-banner bookkeeping from the
  time-sensitive hook path.
- Improved alert matching, notification deduplication, config preservation,
  legacy-hook repair, and Windows behavior.
- Expanded regression coverage for Antigravity, Codex, Claude events, tmux,
  persistent alerts, snooze, channel delivery, and configuration safety.

### Installation and distribution

- Pointed all distribution and update URLs at this fork and made the curl/irm
  installers the supported installation path.
- Reworked the Unix installer to stage and validate a GitHub source tarball,
  then atomically swap it into place without damaging an existing install.
- Added `CODE_NOTIFY_REF` support for installing a branch, tag, or commit SHA.
- Added source-only, CalVer GitHub Releases for pinning known versions without
  publishing npm or Homebrew packages.
